# Инструкция по добавлению расписания

## Структура файлов
- `september.js` - расписание на сентябрь 2025
- `october.js` - расписание на октябрь 2025
- `month-template.js` - шаблон для создания новых месяцев

## Как добавить новый месяц:

1. **Скопируйте** `month-template.js` и переименуйте в название месяца (например, `november.js`)

2. **Измените переменную** в начале файла:
   ```javascript
   const november2025 = {
   ```

3. **Добавьте даты и занятия** в формате:
   ```javascript
   "2025-11-15": [
       {
           time: "08:30 - 10:00",
           subject: "Математический анализ",
           teacher: "Иванов И.И.",
           room: "101",
           type: "lecture"
       }
   ]
   ```

4. **Подключите файл** в `index.html`, добавив строку:
   ```html
   <script src="november.js"></script>
   ```

5. **Обновите** `script.js`, добавив проверку для нового месяца:
   ```javascript
   if (typeof november2025 !== 'undefined') {
       allScheduleData = { ...allScheduleData, ...november2025 };
   }
   ```

## Типы занятий:
- `"lecture"` - Лекция
- `"seminar"` - Семинар
- `"lab"` - Лабораторная работа
- `"practice"` - Практическое занятие

## Формат даты:
Всегда используйте формат `"ГГГГ-ММ-ДД"` (например, `"2025-11-15"`)

## Формат времени:
Используйте формат `"ЧЧ:ММ - ЧЧ:ММ"` (например, `"08:30 - 10:00"`)
